# Dead Reckoning
![icon](docs/atlas_hud_icon.png)

(formerly __Antique Atlas Compass HUD__ )

Displays a compass ribbon in the HUD with __Surveyor__ map markers.

## Features
* Displays a compass ribbon with directions in the HUD.
* Displays colored Surveyor markers on the ribbon.
* Supports __Antique Atlas 4__, __Hoofprint__, and any other mod that uses Surveyor for waypoints.
* All elements of the HUD can be toggled individually.
* Can optionally require the player to have a compass item in hand, in the hotbar, or in the inventory to display the ribbon.
